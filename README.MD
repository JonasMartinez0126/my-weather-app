```sql
CREATE TABLE IF NOT EXISTS weather_history (
  id SERIAL PRIMARY KEY,
  ciudad VARCHAR(100),
  temperatura DECIMAL(5,2),
  clima VARCHAR(100),
  fecha TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

Ejecutar backend:

```bash
npm run dev
```

Servidor corriendo en `http://localhost:4000/api`.
